# PR Gate Checklist

Use when GitHub is enabled for the project.

- [ ] Task card has branch, GitHub issue, GitHub PR, and CI status when available.
- [ ] PR title names the task ID and user-visible change.
- [ ] PR body includes goal, changed files, verification commands, risks, and rollback notes.
- [ ] PR body includes command risk classifications for approval-required, forbidden, or unclear commands.
- [ ] CI checks are passing or failures are explained and approved.
- [ ] PR diff matches the task card boundaries.
- [ ] No unrelated files or generated noise are included.
- [ ] Reviewer result is pass, request changes, or blocked.
